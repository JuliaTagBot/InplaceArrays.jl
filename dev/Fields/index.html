Interface

InplaceArrays.Fields.FieldLike — Type.
abstract type FieldLike

Abstract type representing either a field ( for N==1) or a basis of fields (for N==2) of value T, evaluable at points with D components.

The following functions need to be overloaded for derived types:

  • evaluate!(cache,f::FieldLike,x)
  • field_cache(f::FieldLike)
  • num_dofs(f::Basis)

The following functions can optionally be also provided

  • gradient(f::FieldLike)

The interface can be tested with these functions

  • test_field
  • test_basis
